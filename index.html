Common CSS Properties

As we know, there is a lot of properties in CSS, but now we are covering a few of them in this section, we'll start to look more in-depth at common CSS properties that you'll use to style your HTML elements.

Most Common Properties

Tag Name Description

color:

 Color Set the text-color for different elements

font-size:

 Font Size Set the font size for different elements

background-color:

 Background Color Set the background color for a page

background:

 Background Set different background properties in one declaration

border:

 Border Set the style of the borders for different elements

Another Common Properties

Tag Name Description

height: / width:

 Height/Width Specify the height and width of a HTML element respectively

background-image:

 Background Image Allows us to add a background image to an HTML element

border-radius:

 Border Radius Property allows you to add rounded corners of a HTML elements.
